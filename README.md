# dragon-ball
 A random 2d game create by Unity

# Editor version
 2022.1.24f1

# Exe
 There are executable releases in Releases section
# Controls
Frog : ← ↑ →<br>
Dragon : WASD<br>
R: reset players and ball positions<br>
Space: open/close menu
